## Discovery (plan did not flag) ‚Äî dockerfilesDir injection

`dockerfilesDir()`/`dockerfilePath()` resolved **module-relative**
(`../../dockerfiles`). Dockerfiles are **consumer-owned**; base has none ‚Äî so a
vendored base would resolve into base (wrong). Resolved by taking
`opts.dockerfilesDir` (absolute path or thunk), mirroring the deps-injection seam.
**Carry the same contract into the step-2 driver** (it calls `mounts.dockerfilesDir()`
at build time). Note: `FUTURE_WORK/migrate/260623-dockerfiles-dir-injection-seam.md`.
